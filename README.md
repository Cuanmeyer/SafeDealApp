# SafeDealApp
Android and IOS Xamarin applications. Functionality added: Login &amp; Sign Up Screens,custom animation used for fragment, Access Phone Camera, Storage, Gallery, Calling, Maps, Displaying Markers on Maps, Saving to Azure database, Uploading image to blob storage and displaying it, Sliding layout with changing colour tabs
